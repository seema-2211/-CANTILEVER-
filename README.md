# Chatbot with NLP

## Overview

This project is an NLP-based chatbot developed using Python, Word2Vec, LSTM, and Flask.

The chatbot is trained on the Cornell Movie Dialog Corpus and uses Natural Language Processing techniques to preprocess text, generate embeddings, and generate conversational responses through a web interface.

## Features

* Text preprocessing and cleaning
* Tokenization using Keras
* Word2Vec embeddings using Gensim
* Seq2Seq LSTM chatbot model
* TF-IDF based response retrieval
* Flask web interface

## Technologies Used

* Python
* TensorFlow / Keras
* Gensim
* Scikit-learn
* Flask
* NumPy
